# AI-Powered Inventory Management System

An end-to-end AI-powered Inventory Management System built using Python, PostgreSQL, SQLAlchemy, and Streamlit. 
The project demonstrates database design, backend development, dashboard visualization, and will be extended with Machine Learning demand forecasting and an AI-powered inventory assistant.

## Overview

This project is an AI-powered Inventory Management System developed using Python, PostgreSQL, SQLAlchemy, and Streamlit.
The application helps businesses manage products, inventory, and sales while providing real-time dashboard analytics.
Future versions will include demand forecasting using Machine Learning and an AI Assistant powered by Large Language Models.
